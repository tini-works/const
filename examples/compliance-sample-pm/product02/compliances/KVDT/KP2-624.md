### Requirement

The software system provides the user with the ability to capture the type of disease within the scope of service documentation and to transmit it with the billing.

### Rationale

The requirement results from the EBM billing provisions for GOPen 11233, 11511 to 11513, 11516 to 11518, 11521, 11522, 19421 and 19424, 19451 to 19453 and 19456.

### Acceptance Criteria

1. The software enables the user to specify the "Type of disease" (FK 5079) as free text within the scope of service documentation.
2. The software transmits with the billing the "Type of disease" specified by the user for a service in field 5079.
3. If the user makes multiple entries (count >1) for "Type of disease", field 5079 must also be transmitted multiple times (count >1) in the billing file.
