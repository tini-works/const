### Requirement

A **substitute value** is a field content that is only to be transmitted when there is actually **no** value for a mandatory field listed in the table present in the practice. A substitute value is not to be confused with a **default value**. Substitute values for the fields listed below are defined in Chapter 7 "Feldverzeichnis" of the KVDT data record description [KBV_ITA_VGEX_Datensatzbeschreibung_KVDT].

The fields listed in Table 1 must **not** be automatically populated with substitute values by default -- i.e., not automatically without individual confirmation.

| Geburtsdatum (Date of Birth) | 3103 |
