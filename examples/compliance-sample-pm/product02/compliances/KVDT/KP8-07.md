### Requirement

The software must inform the user that services will not be reimbursed as a Hybrid-DRG service if a value greater than "0" is entered in field 5030 (Anesthesia hours) and transferred to the billing file.

### Rationale

Services that contain anesthesia hours are not grouped into a Hybrid-DRG, as in these cases it is a DRG service. Therefore, such services cannot be billed and reimbursed as Hybrid-DRGs.

### Acceptance Criteria

1. If, when recording Hybrid-DRG services, the user enters a value > "0" in field 5030 (Anesthesia hours), the software must display a warning message covering the following points:
   a) Hybrid-DRG services that include anesthesia hours will not be reimbursed as a Hybrid-DRG service, because in that case no Hybrid-DRG applies.
   b) Anesthesia times that occur during anesthesia are not to be specified in the billing.

### Condition

Implementation of Hybrid-DRG billing according to the record type HDRG.
