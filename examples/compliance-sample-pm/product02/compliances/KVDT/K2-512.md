### Requirement

The software can provide the user with the functions based on Kapitel 4 "Umsetzung der Spezifikation - 116117 Terminservice TSS-Abrechnungsinformation" of the Anforderungskatalog Terminservice.

### Rationale

Due to the separate remuneration for treatments resulting from Terminvermittlungen through the Terminservice, the billed services are to be marked by Schein and the relevant information is to be transmitted per the defined requirements.

### Acceptance Criteria

1. The software must implement the functions per Kapitel 4 "Umsetzung der Spezifikation - 116117 Terminservice Abrechnungsinformation" of the Anforderungsdokument "Anforderungskatalog Terminservice" in version 3.0.0 [KBV_ITA_VGEX_Anforderungskatalog_116117_TSS] and provide them to the user during patient capture, practice management, and billing.
2. The software must demonstrate the implementation of functionalities within the KBV Zertifizierungsverfahren "116117 Terminservice Abrechnungsinformation".
3. The software must have successfully completed the audit for the application "116117 Terminservice Abrechnungsinformation" at kv.digital. As proof, the audit certificate must be submitted to KBV within the certification process "116117 Terminservice Abrechnungsinformation".

### Notes

Implementation is only meaningful for systems with Arzt-Patienten-Kontakt.
