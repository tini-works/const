### Requirement

The software can provide the user with the functions based on the KBV-Webservice "Kollegensuche" for physician search purposes.

### Rationale

Physicians should be supported in searching for other physicians. The KBV provides the Webservice "Kollegensuche" for this purpose.

### Acceptance Criteria

1. The software uses the Webservice-Kollegensuche according to the document:
   - "Webservice-Kollegensuche der KBV" in the current version [KBV_ITA_VGEX_SST_Kollegensuche]
2. The software fulfills the following requirements:
   - [KP2-507](KP2-507.md)
   - [KP2-508](KP2-508.md)
   - [KP2-509](KP2-509.md)
