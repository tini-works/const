### Requirement

The software must enable the user to identify GOPen within the scope of service documentation with a "(Branch) Practice site number of the place of service delivery" (BSNR) and a "Lifelong physician number of the service provider" (LANR).

### Rationale

In implementation of the Contract Physician Law Amendment Act, every billed service must be identified with the "(Branch) Practice site number of the place of service delivery" and the "Lifelong physician number of the service provider".

In the context of ASV billing by hospital physicians, the physician identification can be done with a "Pseudo-LANR for hospital physicians", cf. § 2 Nr. 25 ASV-AV in conjunction with Appendix 3 ASV-AV.

### Acceptance Criteria

1. The software must enable the user to identify GOPen captured within the scope of service documentation with a practice site number and a lifelong physician number.
2. The software transmits with the billing the (branch) practice site number specified for the GOP in field FK 5098 and the LANR in field FK 5099 or the Pseudo-LANR in field 5101.
