### Requirement

1. The 9-digit Institutionskennzeichen (IK) of the insurance card must be used as a search key for a cost unit, regardless of whether
   a) the IK was read via the insurance card, or
   b) was manually entered (e.g. in the Ersatzverfahren) (see function [P2-410](P2-410.md)).
2. If the IK is present in the KT-Stammdatei (under element `/kostentraeger/ik_liste/ik/@V`), then a Kostenträger-Stammsatz (`/kostentraeger/@V` of the KT-Stammdatei(=VKNR)) must be uniquely identifiable.
3. The VKNR from the Krankenversichertenkarte must not be used for billing.
4. If an element `////AbrechnenderKostentraeger/Kostentraegerkennung` exists on the eGK, this IK must be used as the mandatory search key for a cost unit (see "Mappingtabelle_KVK" [KBV_ITA_VGEX_Mapping_KVK]) and for further processing and issuance of contractual physician forms according to the following case distinctions.
