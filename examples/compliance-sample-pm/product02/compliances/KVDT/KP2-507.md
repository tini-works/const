### Requirement

The software provides the user with search functions based on the data in the "Kollegensuche".

### Rationale

Physicians should be supported in searching for other physicians. The KBV provides the Webservice "Kollegensuche" for this purpose.

### Acceptance Criteria

1. The software provides the user with at least the search functions named in Kapitel 6 "Festlegungen für das PVS" of the document [KBV_ITA_VGEX_SST_Kollegensuche].
2. The software must display the search results to the user in a suitable manner.
   a) The software must provide the user with the ability to view all information for a data record.
