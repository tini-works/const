### Requirement

The software enables printing of the PDF template for Form "PTV 3".

### Rationale

The user must have the ability to access and print the document via the software.

### Acceptance Criteria

1. The user can print the PDF template of Form "PTV 3" [EXT_ITA_AHEX_PTV3] directly from the system.
