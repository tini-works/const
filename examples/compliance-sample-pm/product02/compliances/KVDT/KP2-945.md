### Requirement

The software enables access to the PDF template for the "Patient Declaration European Health Insurance" directly from the system.

### Rationale

The user must have the ability to access and print the document via the software.

### Acceptance Criteria

1. The user can access the PDF template ([EXT_ITA_AHEX_Erklaerung_EHIC_PEB]) of the "Patient Declaration European Health Insurance" directly from the software as needed.
