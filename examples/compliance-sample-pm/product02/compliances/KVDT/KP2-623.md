## KP2-623 — Use of HGNC Gene Symbol Chains for Billing Documentation

| **Matched by** | — |

### Requirement

The software system provides the user with the ability to use defined HGNC gene symbol chains within the scope of service documentation.

### Rationale

Through the use of HGNC gene symbol chains, the effort for billing documentation of extensive genetic examinations shall be reduced.

### Acceptance Criteria

1. Only HGNC gene symbol chains that were defined according to requirement [KP2-622](KP2-622.md) may be used.
2. The user must be able to assign an HGNC gene symbol chain to a GOP to be billed. The software system must automatically transmit the individual contained HGNC gene symbols each in a field 5077.
3. After adopting an HGNC gene symbol chain into the service documentation, the user has the ability to add further HGNC gene symbols (according to requirement [KP2-621](KP2-621.md)) and/or further HGNC gene symbol chains to a GOP to be billed.
