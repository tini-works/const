### Requirement

By default, billing is practice-site-based. The filename of the KVDT billing file is formed from the practice site number (FK 0201, besa data record) (see KVDT data record description [KBV_ITA_VGEX_Datensatzbeschreibung_KVDT]).

### Notes

Deviating from the default case, the software may allow the user to export partial billings based on additional criteria such as LANR (FK 5099), GOPen (FK 5001), VKNR+KTAB combination, etc.
