### Requirement

The software must provide the user with a function for transmitting the online billing based on KIM.

### Rationale

Starting with the billing of the first quarter of 2024, 1Click via KIM can be used for transmitting the billing, provided the respective Kassenärztliche Vereinigung supports the procedure.

### Acceptance Criteria

1. The software provides the user with the functions for billing starting from the first quarter of 2024 in accordance with the following requirements document:
   a) "1ClickAbrechnung V2.1" in the latest current version [Spezifikation_1Click_KIM].
2. The software must successfully complete the gematik confirmation procedure pursuant to [gematik Implementierungsleitfaden Primärsysteme Telematikinfrastruktur] and submit the confirmation letter -- Bestätigung der Konformität des Primärsystems zur Konnektorschnittstelle: Funktionsumfang KIM -- to the KBV as part of the "1ClickAbrechnung" certification.
