### Requirement

The use of the change file to the KT master data file, made available by the KBV on its website as needed, is optional.

1. If a billing VKNR transmitted with the change file under `/kostentraeger/@V` is **not present** in the user's KT master data file, the corresponding change data record of the change file is added to the KT master data file.
2. If a billing VKNR transmitted with the change file under `/kostentraeger/@V` is **present** in the user's KT master data file, the corresponding KT master record is overwritten (replaced) by the change data record.
