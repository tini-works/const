### Requirement

1. The use of the respectively current valid KT master data file **must** occur in connection with the quarterly billing and the printing/issuance of contractual physician forms. Through appropriate organizational measures, it must be ensured that the user can use the currently valid cost carrier master data file in their billing software in a timely manner at the beginning of each quarter.
2. **Update to the KT master data file:** With an update to the KT master data file, the new KT master data file must be deployed no later than the beginning of the new quarter, even if the billing of the previous quarter has not yet been completed.
