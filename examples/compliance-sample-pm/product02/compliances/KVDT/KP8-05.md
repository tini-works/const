### Requirement

The software must provide the user with a function for transmitting the Hybrid-DRG billing via KIM.

### Rationale

Starting from the billing of the first quarter of 2025, 1ClickHybridDRG can be used via KIM for the transmission of Hybrid-DRG billing, provided that the respective Kassenärztliche Vereinigung supports the procedure.

### Acceptance Criteria

1. The software provides the user with the functions for billing Hybrid-DRG services in accordance with the following requirements document:
   a) "1ClickHybridDRG" in the latest current version [Spezifikation_1ClickHybridDRG].

### Condition

Implementation of Hybrid-DRG billing according to the record type HDRG.
