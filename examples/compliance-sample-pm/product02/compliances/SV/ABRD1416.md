### Requirement

OPS code 5035 must be required for specialty service codes E3a/A3a/E3b/A3b/E4a/A4a/E4b/A4b/E5a/A5a/E5b/A5b

### Acceptance Criteria

1. Given a specialty code (E3a, A3a, etc.) documented, when OPS 5035 is missing, then validation blocks with a mandatory-OPS error
