### Requirement

During a full import (Vollimport), all data from the PTV must be automatically transferred to the Vertragssoftware if the patient can be uniquely matched by eGK-Nummer; defined exception cases (unmatched eGK, status conflicts) must be handled separately with user prompts

### Acceptance Criteria

1. Given a Vollimport with patient data, when patients match by eGK-Nummer, then data is auto-transferred
2. Given an exception case (no eGK match, status conflict), then the system prompts the user for manual resolution
