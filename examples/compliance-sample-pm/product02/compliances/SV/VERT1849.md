## VERT1849 — If the participation status cannot be determined because the patient...

| Field | Value |
|-------|-------|
| **ID** | VERT1849 |
| **Type** | Mandatory |
| **Source** | AKA Q1-26-1 |
| **Section** | 3.3 VERT — Contract Participation |
| **Status** | TBC |
| **Goals** | BG-1a |
| **Verification Method** | Integration test (API) |
| Matched by | [US-VERT1849](../../user-stories/US-VERT1849.md) |

### Requirement

If the participation status cannot be determined because the patient has no valid eGK-Nummer, the Vertragssoftware must display: 'The participation status cannot be determined because no valid eGK Versichertennummer is available.' This hint must only appear when the status is actively being determined

### Acceptance Criteria

1. Given a patient without eGK-Nummer, when participation status is being determined, then the hint 'Der Teilnahmestatus kann nicht ermittelt werden, da keine gueltige eGK-Versichertennummer vorliegt' is displayed
2. Given the status is not being actively queried, then the hint does not appear
