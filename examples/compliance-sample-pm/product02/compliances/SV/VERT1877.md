### Requirement

After a standard import, the Vertragssoftware must display an import protocol showing: contract of the imported Patiententeilnehmerverzeichnis, VP-ID of the Vertragspartner, quarter/year, importing user, import type (Standardimport), timestamp, and detailed import results

### Acceptance Criteria

1. Given a standard import completes, when the protocol is displayed, then it shows: contract, VP-ID, quarter/year, user, type (Standardimport), timestamp, and itemized results
