### Requirement

The Pruef- und Abrechnungsmodul returns Priscus-Liste markings for insurance-specific medication recommendations, which must be displayed to the user in column form

### Acceptance Criteria

1. Given insurance-specific medication recommendations, when displayed, then a Priscus-Liste column is shown for each medication
