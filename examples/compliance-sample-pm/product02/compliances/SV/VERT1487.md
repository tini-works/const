### Requirement

The Vertragssoftware must prevent importing a Patiententeilnehmerverzeichnis if a more recent quarter has already been imported for the same Vertragspartner and Vertrag

### Acceptance Criteria

1. Given a Patiententeilnehmerverzeichnis for a newer quarter already imported, when the user attempts to import an older quarter, then the import is blocked
