### Requirement

The Vertragssoftware must provide a function to print the 'Merkblatt Versicherter Hilfsmittel' document per AKA-Basisdatei for steuerbare Hilfsmittel prescriptions

### Acceptance Criteria

1. Given a steuerbare Hilfsmittel prescription, when the user requests the Merkblatt, then the Merkblatt Versicherter Hilfsmittel per AKA-Basisdatei is printed
