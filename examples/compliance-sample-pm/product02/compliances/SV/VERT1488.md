### Requirement

The Vertragssoftware must provide an overview of all previously imported Patiententeilnehmerverzeichnisse showing contract, physician name/LANR, quarter/year, importing user, import timestamp, and link to the import protocol

### Acceptance Criteria

1. Given imported Patiententeilnehmerverzeichnisse exist, when the user opens the overview, then each entry shows contract, physician name/LANR, quarter/year, importing user, import date/time, and protocol link
