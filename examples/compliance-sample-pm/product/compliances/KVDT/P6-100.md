### Requirement

1. Through appropriate organizational measures, it must be ensured that the user has the currently valid KV-Specifika master data file available for the billing software in a timely manner at the beginning of each quarter.
2. All KV-specific data of the respectively responsible KV must be available to the user in the billing system.
3. The billing software may only grant the user access to the specifications of the responsible Kassenärztliche Vereinigung.
