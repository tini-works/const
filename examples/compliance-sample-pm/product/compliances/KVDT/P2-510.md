### Requirement

The following billing-preparatory functions must be included in the billing software:

1. The execution of **trial billings** (Probeabrechnungen) must be possible for the user at any time.
2. The creation of **daily control lists** (Tageskontrolllisten) must be possible for the user at any time. Daily control lists contain an overview of physician-patient contacts on a calendar day with data on patient personal details, services rendered, and diagnoses.
3. The user must have the ability to **correct billing step by step** -- or across the entire quarter -- as needed.
