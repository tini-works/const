### Requirement

The software must offer the user a function so that the digital form created according to requirement [K26-03](K26-03.md) can be sent via a secure transmission channel.

### Rationale

To adequately support the physician in their work and to protect sensitive data from unauthorized access, the system must enable the user to transmit the digital form via a secure transmission channel.

### Acceptance Criteria

1. The user is enabled to send the created digital form via a secure transmission channel (see [K26-05](K26-05.md)).
2. A corresponding implementation of this function has been confirmed by the software manufacturer as part of the supplementary declaration.
