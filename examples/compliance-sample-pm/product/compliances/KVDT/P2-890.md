### Requirement

A uniform design of the patient receipt is targeted. The patient receipt must implement the following information and layout specifications:

| Subject / Field Name | Description / Specification / Formula |
|---------------------|--------------------------------------|
| Paper format | DIN A4 |
| Font type | Any monospace fonts |
| Font size | 12 CPI |
| Line spacing | Single-spaced |
| Address field | Structure and position of patient address per DIN 5008 |
| Health insurance name | Health insurance name for printing (see KTS) |
| Insured number | |
| Sender | Practice address / Physician stamp |
| Issue date | Date of the day |
| Subject | Service and cost information |
| Explanatory text | Dear patient, for your information, you will find below a listing of the medical services that were rendered for you in the period stated below, and about the treatment costs that can be provisionally claimed as physician's fee. The treatment costs have been settled through payments from your health insurance fund. This is not an invoice. |
| Text for quotation (standard text) | The amount stated below for the medical services I have rendered will, due to the limitation of the health insurance funds' financial resources, potentially only be partially paid to me. The payment will subsequently be reduced by the health insurance fund to the extent that the funds made available by your health insurance fund are sufficient. |
| or when entering X: | |
| Text for quotation (quota known) | The amount stated below for the medical services I have rendered will, due to the limitation of the health insurance funds' financial resources, potentially only be partially paid to me. The payment will subsequently be reduced by the health insurance fund by X %, so that the funds made available by your health insurance fund are sufficient. |
| Treatment period | Treatment date or treatment quarter |
| Point value | (per [P2-830](P2-830.md)) |
| Service listing | In tabular form: Day / GNR / Short description / Points / Fee. Exceptions: For EBM services where no points are stored in the GO-Stammdatei, the entry in the Points column can be omitted or replaced by "-". When the patient receipt contains only EBM services for which no points are stored in the GO-Stammdatei, the following alternative table format may also be used: Day / GNR / Short description / Fee |
| Summary line | Costs for medical services in EUR |
| Reimbursement by your health insurance in EUR | **K * (100% - X)** where X = share of non-reimbursed services, K = costs for medical services in EUR |
| Footnotes | (see example) |
| Page numbering | Continuous |
| On continuation pages | Name and issue date in header |
