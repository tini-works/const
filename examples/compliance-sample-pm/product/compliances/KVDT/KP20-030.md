### Requirement

If the billing software is used by specialist groups that could potentially use pnSD/uu, this function is mandatory.

A dialog/configuration option must exist to document whether the use of unit-use reagents (no, exclusively, partially) is recorded at the practice, and this must be permanently stored and editable.

### Clarification

The implementation of this function may be irrelevant for pure LIS systems and need not be mandatorily implemented (hence implemented as a conditional mandatory function). In the RVSA data record, FK 0301 is then to be standardly transmitted with the content "0."
