### Requirement

The system date is fundamentally provided by the operating system. A modification of the system date by the user violates the principles of proper data processing. The provision of a function in the application software that enables a modification of the system date is inadmissible.

### Notes

Every user with system administrator rights has the ability to change the system date. Against this background, it is not expected that the system date must be immutable by the physician. This requirement would be beyond practical reality and would presuppose the use of a radio clock-fed time module in the practice computer. Rather, emphasis is placed on the fact that the application software does not offer system date modification functions at various points that can be activated at will by any user. Normally, a date change should only be possible at the operating system level and only for appropriately authorized persons.
