### Requirement

The software must perform an automatic pre-population of the fee schedule (Gebührenordnung) (FK 4121) to be entered by the user.

### Rationale

Incorrect fee schedule entries lead to problems in the billing process. The entered fee schedule must correspond to that assigned to the cost carrier, against whose charges the billed services are claimed. The pre-population with the corresponding value from the cost carrier master data file supports the user in entering correct data.

### Acceptance Criteria

1. The software must automatically pre-populate the fee schedule (FK 4121) for the ADT billing. It must pre-populate the value of the attribute `/kostentraeger/gebuehrenordnung/@V` of the cost carrier whose attribute `/kostentraeger/@V` corresponds to the determined billing VKNR (FK 4104).
2. The software allows the user to adjust the pre-populated value of the fee schedule (FK 4121) before adopting the billing data record data into the ADT billing.
