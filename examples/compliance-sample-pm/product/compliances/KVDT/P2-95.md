### Requirement

The user must be supported by the software house or the system as follows to more easily use the online billing services of the KVen:

1. All documentation related to the PVS, e.g. user manual, online help, must be supplemented with information about where in the file system the encrypted KVDT billing file is stored.
2. The user must additionally be supported system-side by providing a function for locating the encrypted billing file in the file system. This function should optimally support the user, e.g. by directly navigating to the billing file or invoking a file explorer with a correctly pre-set path or link. Alternatively, a "function" that enables the user to directly access the corresponding information from the online help, in accordance with item a), would also be sufficient.
