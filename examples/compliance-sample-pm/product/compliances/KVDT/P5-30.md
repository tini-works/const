### Requirement

If the stand-alone version of the XPM provided by the KBV is used, it must be ensured that the user has access at the file system level to the unvalidated and unencrypted billing file generated by the PVS after the billing run.

**Annotation:** The billing file generated by the PVS is needed as input for the stand-alone version XPM.
