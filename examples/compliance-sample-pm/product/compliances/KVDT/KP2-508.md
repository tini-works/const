### Requirement

The software enables the user to adopt a Betriebsstättennummer from a search result in the "Kollegensuche" into the billing.

### Rationale

Physicians should be supported in searching for other physicians. The KBV provides the Webservice "Kollegensuche" for this purpose.

### Acceptance Criteria

1. The software must allow the user to specify the Betriebsstättennummer from a data record of search results per [KP2-507](KP2-507.md) or from the Favoritenliste per [KP2-509](KP2-509.md) in the billing as rationale for a GOP per [KP2-511](KP2-511.md).
