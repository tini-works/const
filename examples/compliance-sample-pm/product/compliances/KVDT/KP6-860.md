### Requirement

The data records of the OPS master data file [SDOPS] must not be changed in terms of content.

### Rationale

Per § 295, Absatz 1 SGB V, it is regulated that the coding of performed operations and other procedures must be done using the respectively valid version of the procedure code. The provided version of the [SDOPS] represents the respectively valid version of the procedure code for application in the scope of § 295 SGB V.

### Acceptance Criteria

1. The software ensures that the data of the OPS master data file [SDOPS] cannot be changed in terms of content by the user.
2. The software ensures that the data of the OPS master data file of the KBV is not changed in terms of content during any necessary transformations into another format, for example.

### Condition

Excluded from the implementation of this requirement are software systems without APK.
