### Requirement

A PVS must manage all physicians practicing at each practice site number (Betriebsstättennummer) with their LANR and transmit this information within the scope of KV billing in the besa data record according to the KVDT data record description [KBV_ITA_VGEX_Datensatzbeschreibung_KVDT]. If a PVS is operated at multiple locations (mobile or networked), a practice site assignment must be implemented in addition to the user notification.
