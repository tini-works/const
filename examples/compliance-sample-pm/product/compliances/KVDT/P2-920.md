### Requirement

If the ICD classification is explicitly required for an EBM service through the GO master file, the billing system must check during service recording whether the required diagnosis is mentioned in FK 6001 or 3673 and, if applicable, display a warning notice indicating the required ICD codes. If none of the required ICD codes is entered, the entry of the EBM code must not be rejected.

### Rationale

To support the user in planning and billing services that are only billable with justification through at least a specific treatment diagnosis, the software points the user during recording to the list of justification-eligible diagnoses.

### Acceptance Criteria

1. If the user records an EBM service with a fee schedule number for which an active list of justification-eligible diagnoses is stored in the EBM master file (SDEBM XML element `../gnr/bedingung/begruendungen_liste/icd_liste` with `../gnr/bedingung/begruendungen_liste/icd_liste/@V='true'`) and if none of the ICD-10-GM codes contained in the list is recorded for the billing fields "ICD Code" or "Permanent Diagnosis (ICD Code)" (FK 6001 or 3673), the billing software must display a warning notice to the user that the service is only billable with justification through at least one of these treatment diagnoses.
2. The software must give the user the option to record the EBM service in the case described in acceptance criterion 1, even if the user has not recorded at least one of the ICD-10-GM codes from the list of justification-eligible diagnoses for the billing fields "ICD Code" or "Permanent Diagnosis (ICD Code)" (FK 6001 or 3673).
