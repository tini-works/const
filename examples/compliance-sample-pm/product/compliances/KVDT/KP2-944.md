### Requirement

The software ensures that upon approval of an applied psychotherapy with GOPs 3550X, 3551X, 3552X, 3553X, 3554X, 3555X, 3570X, and 3571X, the respective actual GOPs according to EBM are transferred in the billing into field 4253 (Approved GOP for the insured) or field 4256 (Approved GOP for the accompanying person).

### Rationale

The amount of the remuneration depends on the number of participants. Consequently, for each psychotherapy procedure as group therapy, there are seven GOPs each for short-term therapy and seven GOPs for long-term therapy. Once a psychotherapy procedure for short-term therapy or long-term therapy is approved, the associated seven GOPs are also approved.

### Acceptance Criteria

1. When a psychotherapy with GOPs 3550X, 3551X, 3552X, 3553X, 3554X, 3555X, 3570X, or 3571X is applied for and approved, by substituting the possible participant counts 3, 4, 5, 6, 7, 8, and 9 for the variable X, seven GOPs are created in each case, which are automatically transferred in field 4253 (Approved GOP for the insured) or field 4256 (Approved GOP for the accompanying person).
2. The user can deactivate the default automatic transfer.
3. The user can manually select from the possible seven GOPs the approved GOPs for each procedure to be transferred.
4. The user has the ability to edit already entered GOPs (delete or add GOPs).
