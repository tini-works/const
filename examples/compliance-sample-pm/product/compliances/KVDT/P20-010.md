## P20-010 — Function for Managing Ring Trial Certificates

| **Goals** | — |
| **Verification Method** | — |

| **Refer** | — |

### Requirement

Every billing software that could be used for billing laboratory services according to the key table [S_NVV_RV_ZERTIFIKAT, OID 1.2.276.0.76.3.1.1.5.2.22] must offer a function that allows the user (with support of the GOPs linked via the key table) to record the RV-relevant analytes and manage the respective certificates.

### Notes

The initial recording could be triggered, for example, when billing laboratory services according to the key table [S_NVV_RV_ZERTIFIKAT, OID 1.2.276.0.76.3.1.1.5.2.22] are identified or when outdated information about RV certificates exists (e.g., reminder function).
