### Requirement

The version of the OPS master data file provided in the current quarter is valid for use starting from the beginning of the following quarter, as long as a new version of the OPS master data file is available.

### Rationale

Per § 295, Absatz 1 SGB V, it is regulated that the coding of performed operations and other procedures must be done using the respectively valid version of the procedure code. The provided version of the SDOPS represents the respectively valid version of the procedure code for application in the scope of § 295 SGB V.

### Acceptance Criteria

1. The software ensures that the OPS master data file delivered by the KBV in the current quarter is deployed starting from the beginning of the following quarter and remains in use as long as a new version of the OPS master data file is available.
2. The software must provide the user with the ability to view the validity status of the integrated master data file.

### Condition

Excluded from the implementation of this requirement are software systems without APK.
