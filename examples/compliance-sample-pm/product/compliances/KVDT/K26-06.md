### Requirement

The software must offer the user a function so that a digital form can be received via a secure transmission channel. The received digital Form 6 fulfills the provisions of the technical handbook for digital preprints (see: [KBV_ITA_VGEX_Technisches_Handbuch_DiMus]).

### Rationale

To adequately support the physician in their work, the system must enable the user to receive the digital form via a secure transmission channel.

### Acceptance Criteria

1. The user is enabled to receive a digital Form 6 via the secure transmission channel (see [K26-05](K26-05.md)).
2. A corresponding implementation of this function has been confirmed by the software manufacturer as part of the supplementary declaration.
