## P2-80 — Capture and Transmission of the AMV Certification Number

| Field | Value |
|-------|-------|
| **ID** | P2-80 |
| **Type** | Mandatory |
| **Source** | KVDT v6.06 |
| **Section** | 2.1.6 Einsatz eines zertifizierten Arzneimittelverordnungssystems gemäß Arzneimittelwirtschaftlichkeitsgesetz (AVWG) |
| **Status** | TBC |
| **Goals** | — |
| **Verification Method** | — |
| **Matched by** | — |
| **Refer** | — |

### Requirement

The billing software:

1. must query the user about the use of AMV-certified software and enable the capture of the KBV certification number of the AMV-certified software for transmission via KVDT (FK 9250), **or**
2. automatically transmits the KBV certification number of the AVS via KVDT (FK 9250), if this is possible.
