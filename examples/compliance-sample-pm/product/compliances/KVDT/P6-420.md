### Requirement

The (update) version of the PLZ master data file of the KBV delivered in the current quarter can already be deployed in the current quarter; however, the deployment of this PLZ master data file must occur no later than the beginning of the following quarter. The delivered version of the PLZ master data file remains valid until a new version is delivered.
