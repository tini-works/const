### Requirement

The software must enable the user to create a Favoritenliste. In the Favoritenliste, the user must have the ability to access the essential data relevant to Terminvermittlung.

### Rationale

Physicians should be supported in searching for other physicians. The KBV provides the Webservice "Kollegensuche" for this purpose.

### Acceptance Criteria

1. The software must enable the user to create a "Favoritenliste".
2. The software must provide the user with the ability to mark individual data records from search results based on requirement [KP2-507](KP2-507.md) as favorites.
