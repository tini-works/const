### Requirement

According to the specifications set out in Chapter 2.2.2.1.7, the KT master data file may be temporarily extended:

1. Since new health insurance fund establishments after the editorial deadline for the applicable following quarter's valid KT master data file can occur, new cost carriers must be able to be added as **temporary cost carrier master records** to the KT master data file, regardless of whether an IK was read from an insurance card or (by analogy with the substitute procedure) manually entered.
2. An IK must not be added to an existing KT master record.
