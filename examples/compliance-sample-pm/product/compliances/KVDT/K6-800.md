### Requirement

If a GNR₀ exists in the EBM master data file, the following recommended check sequence for the categories is suggested:

1. Conditions,
2. KV conditions,
3. Rules.

If multiple rules exist for a GNR, the fundamental EBM principle applies that the physician should be billed as favorably as possible. The order of the rules can therefore also be relevant.

### Notes

EBM supplementary numbers are derived variants of nationwide unified fee numbers (GNR) in the value range 00001-88999, identified by an addition in the value range A-Z (= letter suffix).

The principle applies that all rules and references to a GNR mapped in the EBM master data file are primarily specified with the 5-digit number without letter suffix; however, all supplementary numbers with letter suffix with the same 5-digit number are included. The letter suffix is therefore irrelevant for the rules and references to a number, unless explicitly specified and no rules or references deviating from the 5-digit GNR are specified. This also applies to the implicit reference to the current number in the number-of-times condition.

Whether this principle also generally applies to KV-specific EBM supplementary numbers in the value range 89,000 to 99,999 and additionally KV-specific GNR with the attribute `ehd/body/gnr_liste/gnr/@USE <> 74` should be discussed with the respectively responsible Kassenärztliche Vereinigung before a possible implementation of corresponding GNR checks.
