### Requirement

The prescribing software must enable a free-text prescription.

### Rationale

Not all prescriptions can be represented through the structured prescribing options (e.g., products without a PZN).

### Acceptance Criteria

1. The prescribing software must implement a free-text prescription in accordance with the contents of the Technische Anlage elektronische Arzneimittelverordnung.
2. For printing a paper-based prescription, [**P3-724**](P3-724.md) must be observed.
3. The prescribing software displays a notice to the user that a free-text prescription may only be used when the product to be prescribed does not have a PZN, and assists the user with PZN-based prescribing.
