### Requirement

Medicinal products and other products included in the drug supply pursuant to § 31 SGB V must be searchable by the following attributes, where applicable:

- Trade name (Merkmal 002, 202, 402, 502, 602)
- For medicinal products: Active substances (Merkmal 004 or 020) and active substance combinations (Merkmal 004 or 020)
- ATC code of the ATC classification according to BfArM; at minimum the substance level (7-digit ATC code) and the substance group level (5-digit ATC code) (Merkmal 018, 209, 409, 508, 608)
- Meaning of the ATC codes of the ATC classification according to BfArM (Merkmal 019, 209, 409, 508, 608)
- Manufacturer/provider (Merkmal 017, 208, 408, 507, 607)
- PZN (Merkmal 001, 201, 401, 501, 601)
- G-BA resolution pursuant to § 35a SGB V (Merkmal 061)
- ICD-10-GM code (Merkmal 136)
- ICD-10-GM name (Merkmal 137)
- Merkmal approval/registration number present yes/no (Merkmal 021)

A combined query is possible. A product or active substance can also be accessed by entering the first letters. A search using the ATC code and the meaning of the ATC code can be performed either by entering the ATC code (7-digit or 5-digit) or by selecting from a tree structure of the ATC classification.

The ICD-10-GM code and the designation of the ICD-10-GM code must be searchable via free-text search; further search options regarding the ICD-10-GM code and the designation of the ICD-10-GM code are possible.
