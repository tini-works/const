### Requirement

The Vertragssoftware must allow managing the URL for the PraCMan-Cockpit to enable launching it from within the software per the AKA-Basisdatei interface specification

### Acceptance Criteria

1. Given the PraCMan-Cockpit URL is configured, when the user manages it, then the URL can be set, updated, and used to launch PraCMan from the software
