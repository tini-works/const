### Requirement

Based on the Blank Prescription Master Data File, the software must recognize in which case a blank prescription is possible. When a blank prescription is possible, the user must be explicitly prompted to decide whether to forgo a blank prescription.

### Rationale

For prescriptions based on indications per § 125a SGB V, the remedy service provider decides on the type of remedy, the duration of therapy, the frequency, and the number of treatment units. The corresponding fees are then not to be charged by the physician (see § 13a of the HeilM-RL). In medically justified cases, even when an indication per § 125a exists, the contract physician may decide on the selection and duration of therapy as well as the frequency of treatment units. A blank prescription is valid for a maximum of 16 weeks, for podological therapy measures a maximum of 40 weeks, from the prescription date. No specific functionality is required in the software for the validity of the blank prescription.

### Acceptance Criteria

1. If the criteria for a blank prescription (SDHMBV XML-Elements ../indikation/erster_icd_code/@V, ../indikation/erster_icd_code/zweiter_icd_code/@V, ../indikation/diagnosegruppe/@V, ../indikation/mindestalter_jahre/@V, and ../indikation/hoechstalter_jahre/@V) exist that match the information on the prescription, the software must prompt the user to decide whether a blank prescription should be issued.
   a. The following notice text is to be displayed:
      "The diagnosis in combination with the diagnosis group meets the criteria for a blank prescription. Should a blank prescription be issued?
      **Yes**, remedy, treatment units, and therapy frequency are determined by the therapist. The prescription is not subject to economic efficiency review.
      **No**, a blank prescription is waived for medical reasons. Information on remedy(ies), treatment units, and therapy frequency are to be determined by the physician."
   b. The age is to be determined based on the prescription date of the respective prescription.
   c. If a blank prescription cannot be issued due to an incomplete or missing date of birth of the patient such that the age condition cannot be unambiguously evaluated (e.g., when the date of birth has no day specification and the age threshold falls in the current month), no blank prescription is to be issued.
   d. An entered code corresponds to a code of the SDHMBV when the value of the entered code is identical to a code value of the SDHMBV or when the code from the SDHMBV represents a left-aligned portion of the entered code. The character "-" is not to be considered when comparing the values of two ICD-10 codes.
2. The user must have the ability at any time during the prescribing process to change the designation of a blank prescription.
3. If the user per (1) has decided that a blank prescription should be issued, the text "BLANKOVERORDNUNG" must be entered in the "Remedy as per Catalog" field. The fields on the prescription for "Treatment Units" and "Therapy Frequency" must not be filled. If these fields are nevertheless filled, termination of the prescribing process (printing or saving) is not permitted after a notice is displayed.
4. When accessing a previous prescription (per [P3-04](P3-04.md)) that was issued as a blank prescription and the criteria for a blank prescription continue to be met, no query per (1) shall occur (subject to point (3)).
5. When accessing a previous prescription (per [P3-04](P3-04.md)) that was not issued as a blank prescription but the criteria for a blank prescription are met, a query per (1) must occur (subject to point (3)).

### Notes

If there is no further criteria beyond an indication entry in the SDHMBV, the indication entry applies to all values of those criteria. For example, if an indication entry contains only a diagnosis group and no ICD-10 code, minimum or maximum age as additional criteria in the SDHMBV, prescriptions with this diagnosis group must be shown the notice text per (1) a. regardless of the specified ICD-10 code and age.

If the user per (1) has decided that a blank prescription should be issued, the following requirements do not apply:

- [P3-21](P3-21.md) Prescription of "priority remedies"
- [P3-13](P3-13.md) Therapy Frequency
- [P3-22](P3-22.md) Additional prescription of a "supplementary remedy"
- [P3-23](P3-23.md) Independent prescription of a "supplementary remedy"
- [P3-24](P3-24.md) Standardized Remedy Combination
- [P3-25](P3-25.md) Sorted Selection List of Remedies
- [P3-11](P3-11.md) Maximum quantity per prescription, guiding treatment quantity
