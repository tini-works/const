### Requirement

The user must be given the option to specify a therapy frequency in the field "Therapiefrequenz".

### Rationale

The physician must, according to § 13 para. 2 lit. i) of the HeilM-RL, provide information on the prescription form regarding the therapy frequency of the remedy treatment. For prescribing measures of nutritional therapy, there are no specifications for the therapy frequency in the Stammdatei of the KBV, since the Heilmittel-Richtlinie of the G-BA makes no provisions regarding therapy frequency. However, the physician should have the option to specify a value on the prescription if deemed necessary.

### Acceptance Criteria

1. The field is to be pre-populated with the guideline value per the frequency recommendation list (SDHM XML-Element ../heilmittelverordnung/frequenzempfehlung_liste/frequenzempfehlung/.. with ../heilmittelverordnung/frequenzempfehlung_liste/frequenzempfehlung/richtlinie/@V equal to true) per the Heilmittel-Stammdatei. The user must be given the option to either select the alternative values from the Heilmittel-Stammdatei (SDHM XML-Element ../heilmittelverordnung/frequenzempfehlung_liste/frequenzempfehlung/.. with ../heilmittelverordnung/frequenzempfehlung_liste/frequenzempfehlung/richtlinie/@V equal to false or ../heilmittelverordnung/frequenzempfehlung_liste/frequenzempfehlung/richtlinie not existent) or to enter deviating values (see point (2)).

    A frequency recommendation represents, depending on the frequency recommendation type, either a number of treatment units "per day" or "per week" or a "number of weeks" between two treatment units.

    The result value of the therapy frequency is to be formatted for printing according to the data from the frequency recommendation list in the Heilmittel-Stammdatei (SDHM XML-Elements ../frequenzempfehlungstyp/@V, ../frequenzempfehlung/minimale_anzahl/@V and optional ../frequenzempfehlung/maximale_anzahl/@V) per the following instructions (Table 2):

    | Formatting instruction | | Frequency recommendation type |
    |---|---|---|
    | Single value | Value range | |
    | "%dx taegl.", \<minimum number\> | "%d-%dx taegl.", \<minimum number\>, \<maximum number\> | Units per day |
    | "%dx woech.", \<minimum number\> | "%d-%dx woech.", \<minimum number\>, \<maximum number\> | Units per week |
    | "alle %d Wochen", \<minimum number\> | "alle %d-%d Wochen", \<minimum number\>, \<maximum number\> | Weeks between two units |

    **Table 2:** Formatting instructions for therapy frequency

2. The user must be able to enter either a single value or a value range. The values represent, analogous to the frequency recommendations, depending on the therapy frequency type, either a number of treatment units "per day" or "per week" or a "number of weeks" between two treatment units. The values may have a maximum of one decimal place. The result is to be formatted according to the instructions given in point (1).
    a. The therapy frequency type of the entry must correspond to the frequency recommendation type predefined in the Heilmittel-Stammdatei (SDHM XML-Element ../frequenzempfehlung_liste/frequenzempfehlungstyp/@V), if a frequency recommendation is defined for the corresponding diagnosis group.
3. For the diagnosis groups SAS and CF (remedy area Ernaehrungstherapie) (SDHM XML-Element ../kapitel/@V equal to "V. Maßnahmen der Ernaehrungstherapie") as well as the diagnosis groups UI1 and UI2 (remedy area Podologische Therapie) (SDHM XML-Element ../kapitel/@V equal to "II. Maßnahmen der Podologischen Therapie"), no frequency recommendations are stored in the Heilmittel-Stammdatei (SDHM XML-Element ../diagnosegruppe/heilmittelverordnung/frequenzempfehlung_liste/). The user must have the option to enter corresponding values per point (2). There is no obligation to fill the field.

### Example

The table below (Table 3) shows some examples of form printing for remedy prescriptions with the six possible therapy frequency specifications.

| Remedy per catalog specification / Heilmittel | Therapy frequency for the remedy treatment |
|---|---|
| Standardisierte Heilmittelkombination(KG;MT;KMT;Waermetherapie) | 1-2x woech. |
| Uebungsbehandlung im Bewegungsbad Gruppe | 1x woech. |
| Stimmtherapie-30 | 1-3x taegl. |
| Stimmtherapie-45 | 1x taegl. |
| Podologische Komplexbehandlung | alle 4-6 Wochen |
| Hornhautabtragung | alle 4 Wochen |

**Table 3:** Example for form printing - therapy frequency specification
