### Requirement

The prescribable remedies are displayed in a sorted selection list.

### Rationale

Due to the large number of prescribable remedies, the Heilmittel-Stammdatei provides a sort order for the display. When displaying the prescribable remedies, the user should see the remedies that are prioritized for selection displayed at the top.

### Acceptance Criteria

1. When displaying the selection lists of remedies, the remedies to be selected are to be displayed in ascending order for the user according to the numerical prioritization per the Heilmittel-Stammdatei (SDHM XML-Element ../vorrangiges_heilmittel_liste/vorrangiges_heilmittel/reihenfolge/@V or ../ergaenzendes_heilmittel_liste/ergaenzendes_heilmittel/reihenfolge/@V).
2. In the case of a "Standardized Remedy Combination", the priority remedies and supplementary remedies are to be placed first in the selection list.
