### Requirement

The user must be given the ability to enter a valid ICD-10-GM code in the "ICD-10-Code" field.

### Rationale

The user must specify a valid ICD-10-GM code on the remedy prescription, except in justified exceptional cases such as a home visit. Entering a second ICD-10-GM code is generally only necessary when a special prescribing need is to apply.

### Acceptance Criteria

1. The user must have the ability in the software to enter an ICD-10-GM code in the "ICD-10-Code" fields with final digit precision.
2. The entry of ICD-10-GM codes must be possible either manually, by adoption from the KBV ICD-10-GM Master Data File (SDICD XML-Element ../diagnose/icd_code/@V), or from the patient record.
   a. The entered ICD-10-GM codes must correspond to the values of the KBV ICD-10-GM Master Data File (SDICD XML-Element ../diagnose/icd_code/@V).
3. In the case of adopting an ICD-10-GM code from the ICD-10-GM Master Data File, the adoption of the ICD-10-GM code into the patient record must be enabled.
4. The user must have the ability in the software to enter a second ICD-10-GM code in the designated field. When entering a second ICD-10-GM code, the software preferentially offers ICD-10-GM codes from the BVB/LHM Master Data File (SDHMA XML-Element ../verordnungsbedarf/heilmittel_liste/heilmittel/sekundaercode/@V or ../verordnungsbedarf/icd_code/@V) for adoption that, in combination with the first entered ICD-10-GM code, define a special prescribing need (SDHMA XML-Element ../verordnungsbedarf/icd_code/@V or ../verordnungsbedarf/heilmittel_liste/heilmittel/sekundaercode/@V). Two entered ICD-10-GM codes define a special prescribing need in combination when, for the same data instance of a care need (SDHMA XML-Element ../verordnungsbedarf), one of the two codes corresponds to the primary code and the other to a secondary code (SDHMA XML-Element ../verordnungsbedarf/icd_code/@V or ../verordnungsbedarf/heilmittel_liste/heilmittel/sekundaercode/@V).
   a. An entered code corresponds to a code of the SDHMA when the value of the entered code is identical to a code value of the SDHMA or when the code from the SDHMA represents a left-aligned portion of the entered code. The character "-" is not to be considered when comparing the values of two ICD-10 codes.
   The software displays the following notice without interrupting the workflow: "Specifying a second ICD-10-GM code is only necessary for establishing certain special prescribing needs."
5. The user must have the ability to use all ICD-10-GM codes from the ICD-10-GM Master Data File.

### Notes

The order of ICD-10-GM codes entered in the form does not need to correspond to the order in the SDHMA.
