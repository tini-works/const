### Requirement

The user must be given the option, for prescriptions within the remedy area "Physiotherapy", to prescribe standardized remedy combinations.

### Rationale

Only for the remedy area Physiotherapy does the remedy catalog provide the option of prescribing a Standardized Remedy Combination. It is possible to prescribe a remedy combination without specifying further details. The remedy service provider then decides, also depending on the course of treatment, on the remedy measures to be applied. The naming of three or more individual remedies may be done by the physician, provided this is medically appropriate.

### Acceptance Criteria

1. The user has the option of selecting a "Standardized Remedy Combination", provided it is eligible for prescription according to the selected diagnosis group per Heilmittel-Stammdatei (SDHM XML-Element ../diagnosegruppe/heilmittelverordnung/standardisierte_heilmittel_kombination/).
2. The user also has the option of selecting only the element "Standardized Remedy Combination". The name of the "Standardized Remedy Combination" is to be written in the first line "Heilmittel". This corresponds to the SDHM XML-Element ../diagnosegruppe/heilmittelverordnung/standardisierte_heilmittel_kombination/name/@V.
3. Additionally, the user has the option, alongside the element "Standardized Remedy Combination", to select at least three priority and/or supplementary remedies of the diagnosis group to combine freely. The number of remedies that can be combined is not limited. This corresponds to the SDHM XML-Elements (../diagnosegruppe/heilmittelverordnung/vorrangiges_heilmittel_liste/vorrangiges_heilmittel/name/@V) and (../diagnosegruppe/heilmittelverordnung/ergaenzendes_heilmittel_liste/ergaenzendes_heilmittel/name/@V). Ending the prescription process (printing or saving) with a selection of fewer than three priority and/or supplementary remedies must not be permitted without a notification display.
4. In the case of a combination per point (3), the names of the "Standardized Remedy Combination" and the individually selected remedies are to be written in the first line "Heilmittel". The name of the "Standardized Remedy Combination" is to be placed first. The names of the individually selected remedies are to be separated by a ";" and placed in parentheses. If the first text field "Heilmittel" is not sufficient, the following lines "Heilmittel" are to be used. The specification of treatment units is only to be defined in the first line.
