### Requirement

The user must be given the option to view the current version of the Remedies Directive of the G-BA (including annexes) [HeilM-RL] as a PDF file within the software.

### Rationale

The user must be given the option to access all information of the Remedies Directive at any time.

### Acceptance Criteria

1. The software must always contain the current Remedies Directive of the G-BA [HeilM-RL] (including annexes) in the form of a PDF file.
2. A link to the G-BA website is not sufficient.
