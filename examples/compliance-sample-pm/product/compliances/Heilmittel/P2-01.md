### Requirement

The data of the valid remedies master data file must be stored in the software for use. The data records of the remedies master data file must not be modified in content.

### Rationale

In the remedies master data file provided by KBV, the data required for a guideline-compliant prescription is made available in electronically readable form. Since the remedies master data file is based on the Remedies Directive of the G-BA, the data must not be altered.

### Acceptance Criteria

1. The remedies master data file from KBV must be integrated into the software and used in the currently valid version.
2. The software provides the user with the option to view the status of the integrated master data file.
3. The data of the remedies master data file must not be modified in content, neither by the software nor by the user.

### Notes

The display of the status of the integrated master data file can be implemented by showing the user the file name or the validity information of the file (SDHM XML Element //ehd/header/service_tmr/@V), indicating which data source the used data originates from.

This requirement does not mandate that software systems must work directly with the format of the provided file. The data of the file may be transferred to another format, e.g., in the form of databases.
