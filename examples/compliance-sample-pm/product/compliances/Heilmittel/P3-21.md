## P3-21 — Prescription of "Priority Remedies"

| Field | Value |
|-------|-------|
| **ID** | P3-21 |
| **Type** | Mandatory |
| **Source** | Heilmittel v2.7 |
| **Section** | 3.2 Verordnung von Heilmitteln |
| **Status** | TBC |
| **Goals** | — |
| **Verification Method** | — |
| **Matched by** | — |
| **Refer** | [P3-07](P3-07.md), [P3-06](P3-06.md) |

### Requirement

The user must be given the ability to populate the "Remedy as per Catalog" fields with one to three priority remedies.

### Rationale

The remedy catalog comprises, depending on the diagnosis group, the various prescribable remedies. Depending on the diagnosis group, up to three priority remedies can be prescribed.

### Acceptance Criteria

1. The user has the ability to populate the "Remedy as per Catalog" fields with one to three different priority remedies (SDHM XML-Element ../diagnosegruppe/heilmittelverordnung/vorrangiges_heilmittel_liste/vorrangiges_heilmittel/name/@V) according to the selected diagnosis group (see [P3-07](P3-07.md)) (SDHM XML-Element ../diagnosegruppe/heilmittelverordnung/vorrangiges_heilmittel_liste/).
2. Exclusively for the remedy areas "Physiotherapy", "Speech, Language, and Swallowing Therapy" (SDHM XML-Element ../kapitel/@V equals "I. Maßnahmen der Physiotherapie", SDHM XML-Element ../kapitel/@V equals "III. Maßnahmen der Stimm-, Sprech-, Sprach- und Schlucktherapie", or SDHM XML-Element ../kapitel/@V equals "IV. Maßnahmen der Ergotherapie"), a selection of multiple priority remedies is possible.
3. The user must be given the ability to select the priority remedies from a list of all available values per point (1).
4. In the remedy area "Podological Therapy", only the remedy whose assigned letter code (SDHM XML-Element ../diagnosegruppe/heilmittelverordnung/vorrangiges_heilmittel_liste/vorrangiges_heilmittel/erforderliche_leitsymptomatik/@V) corresponds to the selected lead symptomatology (SDHM XML-Element ../diagnosegruppe/leitsymptomatik_liste/leitsymptomatik/@V) is selectable.
5. When selecting multiple priority remedies, these must each be assigned to individual form line fields. The "Supplementary Remedy" field is not to be used for this purpose.
6. The user has the ability to enter free text for further specification of the selected remedy (e.g.: [Remedy] as double treatment).
   a. The software must prevent text entries that exceed the space available on the form from being printed into subsequent form line fields that are reserved for the specification of additional remedies.
      Note: The software may notify the user that additional text entries can be made in the "Therapy Goals / further medical findings and notes" field. Alternatively, the overflowing text can be automatically adopted into this field.
7. The prescription of a remedy may be restricted by a required minimum or maximum age of the patient. Remedies for which the patient's age as specified in the Remedy Master Data File (SDHM XML-Element ../vorrangiges_heilmittel_liste/vorrangiges_heilmittel/hoechstalter_jahre/@V) exceeds or the minimum age specified in the Remedy Master Data File (SDHM XML-Element ../vorrangiges_heilmittel_liste/vorrangiges_heilmittel/mindestalter_jahre/@V) is not yet reached must not be prescribed and therefore must not be available for selection.
   a. The age is to be determined based on the prescription date of the respective prescription.
   b. If a check per (7) cannot be performed due to an incomplete or missing date of birth of the patient such that the age condition cannot be unambiguously evaluated (e.g., when the date of birth has no day and month specification and the age threshold falls in the current calendar year), the user must also be offered all remedies with age conditions for selection.
8. The prescription of a remedy may be restricted by a required ICD-10 code. Remedies must not be available for selection per point (3) and must not be prescribed when the ICD-10 code(s) per [P3-06](P3-06.md) do not correspond to any of the ICD-10 codes specified in the Remedy Master Data File (SDHM XML-Element ../vorrangiges_heilmittel_liste/vorrangiges_heilmittel/erforderliche_diagnosen/icd_code/@V).
   a. When no required ICD-10 code was selected, termination of the prescribing process (printing or saving) must not be permitted. The following notice text is to be displayed together with the list of required ICD-10 codes (SDHM XML-Element ../vorrangiges_heilmittel_liste/vorrangiges_heilmittel/erforderliche_diagnosen/icd_code/@V):
      "Only with the following diagnoses with ICD-10 codes is the prescription of the selected remedy possible."
