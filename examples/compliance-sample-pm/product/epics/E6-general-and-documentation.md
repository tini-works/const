## E6: General & Documentation

**Goal:** Meet non-technical compliance requirements — user documentation, advertising rules, and operational guidelines — so that the system passes AKA and KBV audits on documentation and process criteria.

**Origin:** AKA Q1-26-1 catalog section 3.7 (ALLG) — 25 obligations covering user manuals, release documentation, advertising compliance, and support requirements; AVWG v5.8 sections 1.3 and 3.3 — 7 obligations covering user manual, settings backup, and advertising compliance.

**Scope:**
- User manual and online help covering all contract-specific features
- Release notes and version documentation per quarterly update
- Advertising and marketing material compliance with AKA rules
- Support and hotline availability requirements
- Training material for practice staff
- Data protection documentation (GDPR, SGB V)
- System requirements and deployment documentation
- **AVWG user manual** — user manual for prescription software (AVWG 1.3)
- **AVWG advertising controls** — ad-free prescription workflow, advertising display rules, ability to disable advertising (AVWG 3.3)

**Compliance impact:** BG-1a (AKA Certification), BG-1c (KBV AVWG Certification), BG-7 (Regulatory Readiness)
