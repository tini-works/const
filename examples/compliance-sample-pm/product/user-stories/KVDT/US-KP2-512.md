### User Story

As a **Abrechnungsverantwortliche**, I want **to manually populate TSS fields (FK 4103, 4114, 4115) in the billing**,
so that **Terminvermittlung-related billing information can be correctly captured and transmitted for separate remuneration**.

### Compliance Context

Separate remuneration for Terminvermittlung treatments requires that specific TSS data fields are populated in the billing. Manual entry is needed when automated workflows do not apply, ensuring the practice can always fulfill its billing obligations.

### Acceptance Criteria

**Scenario 1: Manual population of TSS fields**
Given the user is editing billing data for a Schein
When the user accesses the TSS data fields
Then the software ensures that the user can manually populate KVDT-Feldkennungen 4103, 4114, and 4115
