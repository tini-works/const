### User Story

As a **Abrechnungsverantwortliche**, I want **to adopt a Betriebsstättennummer from Kollegensuche search results into the billing**,
so that **appointment mediation to a specialist is properly documented with the correct BSNR for billing justification**.

### Compliance Context

Physicians must transmit the Betriebsstättennummer of the specialist when billing for appointment mediation services. The Kollegensuche integration enables accurate data adoption, preventing manual entry errors that could lead to billing rejections.

### Acceptance Criteria

**Scenario 1: BSNR adopted from search results or favorites into billing**
Given the user has search results per [KP2-507] or a Favoritenliste per [KP2-509]
When the user selects a data record
Then the software allows the user to specify the Betriebsstättennummer from that record in the billing as rationale for a GOP per [KP2-511]
