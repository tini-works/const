### User Story

As a **System**, The system must generate and include the RVSA data record in the KVDT billing file (ADT) according to the KVDT data record description,
so that ring trial participation data is electronically transmitted with the quarterly billing.

### Compliance Context

The RVSA data record is a mandatory part of the KVDT billing file for facilities with laboratory services. Non-conformant RVSA records or missing data will cause the billing file to fail validation at the KV level, blocking the entire billing submission.

### Acceptance Criteria

**Scenario 1: RVSA data record generated during billing** [derived]
Given the billing process is running and generating the KVDT billing file (ADT)
When laboratory services with RV participation data exist
Then the RVSA data record is generated in accordance with [KBV_ITA_VGEX_Datensatzbeschreibung_KVDT]

**Scenario 2: RVSA data record included in ADT file** [derived]
Given the RVSA data record has been generated
When the KVDT billing file is assembled
Then the RVSA data record is included in the ADT file per the data record description specifications
