### User Story

As a **Abrechnungsverantwortliche**, I want **to create multiple Satzart 0101/Scheinuntergruppe 00 records in the same quarter for SKT-Versicherte without Versichertenkarte, each with a validity period**,
so that **each paper treatment authorization from the Sozialamt is billed separately with its correct validity period**.

### Compliance Context

SKT-Versicherte without a Versichertenkarte (e.g. Sozialamt patients) may receive multiple paper treatment authorizations per quarter, each with limited validity. Without the ability to create multiple billing records with validity periods (FK 4125), the practice cannot correctly bill for each authorization.

### Acceptance Criteria

**Scenario 1: Multiple Satzart 0101 records for SKT-Versicherte [derived]**
Given an SKT-Versicherte without Versichertenkarte (VKNR-Seriennummer 3rd-5th digit >= 800 or KTAB != 00)
When the user creates billing records in the same quarter
Then the software allows Satzart 0101 with Scheinuntergruppe 00 to be created multiple times
And for each instance, the validity period (FK 4125 Gültigkeitszeitraum von...bis...) must be captured and transferred, provided the validity information is available
