### User Story

As a **Abrechnungsverantwortliche**, I want **to transmit supplementary free-text information when specifying the Vermittlungs-/Kontaktart**,
so that **additional context about the appointment mediation can be included in the billing per TSVG requirements**.

### Compliance Context

Separate remuneration for Terminvermittlungen per § 87a Abs. 3 Satz 5 SGB V may require supplementary information to justify the billing. Without the ability to add free text in FK 4105, relevant details about the mediation context would be lost.

### Acceptance Criteria

**Scenario 1: Supplementary information transmitted with Vermittlungs-/Kontaktart**
Given the user is specifying the Vermittlungs-/Kontaktart (FK 4103)
When the user enters supplementary information as free text
Then the software transmits the supplementary information in FK 4105 to the billing
