### User Story

As a **Vertragsarzt**, I want a tailored selection of potentially relevant RV analytes based on the materials I use,
so that I can document the facility-individual analyte-GOP combinations required for ring trial compliance.

### Compliance Context

The KVDT specification requires that each facility documents which RV analytes it performs, filtered by the materials used. This facility-individual selection drives the RVSA data record content. Without it, the billing file would lack required ring trial participation data.

### Acceptance Criteria

**Scenario 1: Tailored analyte selection based on materials** [derived]
Given the user has specified RV-relevant materials per [P20-021](../../compliances/KVDT/P20-021.md)
When the user accesses the analyte selection
Then the system presents a tailored selection of potentially relevant RV analytes based on the materials used

**Scenario 2: Analyte-GOP selection is persistent and editable** [derived]
Given the user has selected analyte-GOP combinations
When the user saves the selection
Then the selection of analyte-GOP combinations is permanently stored and editable
