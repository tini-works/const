### User Story

As a **Abrechnungsverantwortliche**, I want the AMV certification number to be captured or automatically transmitted in the KVDT billing,
so that the use of a certified drug prescription system is documented per AVWG requirements.

### Compliance Context

The Arzneimittelwirtschaftlichkeitsgesetz (AVWG) requires documentation of whether a certified drug prescription system (AMV) is in use. The KBV certification number must be transmitted via FK 9250 to prove compliance. Without this, the practice cannot demonstrate use of a certified system.

### Acceptance Criteria

**Scenario 1: Query user about AMV-certified software and capture number [derived]**
Given the user prepares billing
When AMV certification information is needed
Then the software queries the user about the use of AMV-certified software
AND enables capture of the KBV certification number for transmission via KVDT (FK 9250)

**Scenario 2: Automatic transmission of AMV certification number [derived]**
Given the AVS certification number is known to the system
When billing is generated
Then the software automatically transmits the KBV certification number via KVDT (FK 9250)
