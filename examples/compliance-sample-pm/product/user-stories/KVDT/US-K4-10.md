### User Story

As a **Abrechnungsverantwortliche** (billing manager), I want the system to restrict billing of pregnancy termination services to the correct SADT record types (sad1, sad2, sad3),
so that pregnancy termination services are billed using only the legally mandated record types.

### Compliance Context

Billing of pregnancy termination services is strictly regulated and must use designated SADT record types: sad1 for outpatient operator services, sad2 for anesthesiologist and follow-up gynecologist services on referral, and sad3 for inpatient physician services. Using incorrect record types would result in rejected claims and potential legal consequences.

### Acceptance Criteria

**Scenario 1: Bill outpatient operator services with sad1** [derived]
Given a pregnancy termination outpatient service is being billed
When the record type is selected
Then the system uses record type "sad1"

**Scenario 2: Bill referred services with sad2** [derived]
Given services of the anesthesiologist (on referral) or follow-up gynecologist are being billed
When the record type is selected
Then the system uses record type "sad2"

**Scenario 3: Bill inpatient services with sad3** [derived]
Given inpatient physician services for pregnancy termination are being billed
When the record type is selected
Then the system uses record type "sad3"
