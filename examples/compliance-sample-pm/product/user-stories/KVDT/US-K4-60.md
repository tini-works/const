### User Story

As a **Abrechnungsverantwortliche** (billing manager), I want the system to automatically derive the cost carrier from plausible Identifier-SA data and allow manual assignment when the identifier is implausible,
so that pregnancy termination cases are always assigned to the correct cost carrier regardless of data quality.

### Compliance Context

Cost carrier assignment for pregnancy termination billing depends on the plausibility of the Identifier-SA. For plausible identifiers, the VKNR is automatically derived from the IK (digits 9-15). For implausible identifiers, automatic derivation is not possible, so the user must manually assign the cost carrier from the authorization certificate. Without this dual approach, some cases could not be billed at all.

### Acceptance Criteria

**Scenario 1: Automatic cost carrier derivation for plausible identifiers**
Given the Identifier-SA is plausible
When the system processes the identifier
Then the VKNR is automatically derived from the IK in digits 9-15 per Kapitel 2.2.2.1

**Scenario 2: Manual cost carrier assignment for implausible identifiers**
Given the Identifier-SA is implausible
When the system cannot automatically derive the cost carrier
Then the user has the option to manually assign the cost carrier
