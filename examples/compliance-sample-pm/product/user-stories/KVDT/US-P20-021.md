### User Story

As a **Vertragsarzt**, I want to select the RV-relevant materials used at my facility from the key table [S_NVV_RV_MATERIAL, OID 1.2.276.0.76.3.1.1.5.2.21],
so that the system can confirm RV participation obligation and filter relevant analyte-GOP combinations.

### Compliance Context

Selection of RV-relevant materials serves two purposes: confirming the RV participation obligation and narrowing down the analyte-GOP combinations for the facility. Without this filtering step, the system cannot correctly determine which ring trial certificates are required, leading to non-compliant billing.

### Acceptance Criteria

**Scenario 1: Material selection dialog available** [derived]
Given the user needs to specify RV-relevant materials
When the user accesses the RV material configuration
Then a dialog/configuration option/selection is available showing RV-relevant materials from [S_NVV_RV_MATERIAL, OID 1.2.276.0.76.3.1.1.5.2.21]

**Scenario 2: Material selection is persistent and editable** [derived]
Given the user has selected RV-relevant materials
When the user saves the selection
Then the selection is permanently stored
And the selection is editable at any time
