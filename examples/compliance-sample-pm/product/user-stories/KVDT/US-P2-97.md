### User Story

As a **Abrechnungsverantwortliche**, I want a function for transmitting billing via KIM (1Click billing),
so that I can submit billing electronically through the TI infrastructure as supported by my KV since Q1 2024.

### Compliance Context

Starting Q1 2024, 1Click billing via KIM enables electronic billing transmission through the telematics infrastructure. The software must implement the "1ClickAbrechnung V2.1" specification and complete the gematik confirmation procedure for KIM. Without this, practices cannot use the increasingly required electronic billing channel.

### Acceptance Criteria

**Scenario 1: Provide 1Click billing via KIM**
Given the user needs to transmit billing
When the user invokes the billing transmission function
Then the software provides 1Click billing functionality per "1ClickAbrechnung V2.1" [Spezifikation_1Click_KIM] in the latest current version

**Scenario 2: gematik confirmation procedure for KIM**
Given the software implements KIM-based billing
When the certification is evaluated
Then the software has successfully completed the gematik confirmation procedure per [gematik Implementierungsleitfaden Primärsysteme TI]
AND the confirmation letter for KIM has been submitted to KBV as part of the "1ClickAbrechnung" certification
