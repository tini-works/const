### User Story

As a **Praxisadministrator**, I want the billing system to provide the currently valid EBM master data (SDEBM) at the beginning of each quarter,
so that service documentation and billing use the correct fee schedule data.

### Compliance Context

The EBM (Einheitlicher Bewertungsmassstab) master data file defines all billable services, their values, and rules. Without the current version, the practice cannot correctly document or bill services, leading to systematic billing errors and rejections across all cases in the quarter.

### Acceptance Criteria

**Scenario 1: Current EBM data available at quarter start** [derived]
Given a new quarter is beginning
When the user accesses the billing system
Then the currently valid data basis of the SDEBM is available within the billing system
