### User Story

As a **System**, The system must restrict ADT billing to only the KT billing areas (KTAB) listed under field identifier 4106 of record type "kvx2",
so that billing uses only authorized cost carrier billing areas defined by the KV.

### Compliance Context

The KV-Specifika master data file defines which KTAB values are valid for ADT billing via record type "kvx2". Using unauthorized billing areas would result in rejected billing records. This constraint ensures compliance with regional KV billing rules.

### Acceptance Criteria

**Scenario 1: Only permitted KTAB values used in ADT billing** [derived]
Given an ADT billing is being generated
When cost carrier billing areas are assigned
Then only the KTAB values listed under field identifier 4106 of record type "kvx2" are used
