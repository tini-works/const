### User Story

As a **Praxisadministrator**, I want **to print the Patient Declaration European Health Insurance form both in full and in parts, with the ADT verification number printed at the defined position**,
so that **the form is produced per Annex 20 BMV-Ae requirements using certified software**.

### Compliance Context

Per Annex 20 BMV-Ae, the Patient Declaration European Health Insurance form must be produced exclusively by certified software with proper positioning of the ADT verification number (PRF.NR.). Non-compliant printing invalidates the declaration.

### Acceptance Criteria

**Scenario 1: Full or partial printing**
Given the user needs to print the Patient Declaration European Health Insurance
When the user initiates printing
Then the user can print the PDF template in full or print parts of the PDF template directly from the system

**Scenario 2: Verification number printed at defined position**
Given the form is being printed
When each page is rendered
Then the software prints the ADT verification number (PRF.NR.) from the KVDT procedure below the contract physician stamp
And the verification number is printed in Arial, font size 5 at the defined position
