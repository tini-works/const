### User Story

As a **Vertragsarzt**, I want **to print the PDF template for Form PTV 3 directly from the system**,
so that **I can produce the psychotherapy form for patient documentation**.

### Compliance Context

Form PTV 3 must be printable from the certified software to fulfill psychotherapy documentation requirements. The ability to print ensures the form is available for patient records and regulatory compliance.

### Acceptance Criteria

**Scenario 1: PTV 3 printable from system**
Given the user needs to print Form PTV 3
When the user initiates printing
Then the PDF template [EXT_ITA_AHEX_PTV3] is printed directly from the system
