### User Story

As a **System**, The system must restrict KVDT file data packages to those listed under field identifiers 9135 and 9138, observing the permitted time frame in 9136 and 9137,
so that only authorized data packages are included in the billing file.

### Compliance Context

The KV-Specifika master data file defines which data packages are permitted for a given KV and time frame. Including unauthorized data packages in the KVDT file would cause validation failures and billing rejections at the KV level.

### Acceptance Criteria

**Scenario 1: Only permitted data packages stored in KVDT file** [derived]
Given a KVDT billing file is being generated
When data packages are written to the file
Then only data packages listed under field identifiers 9135 and 9138 are stored
And the permitted time frame defined in field identifiers 9136 and 9137 is observed
