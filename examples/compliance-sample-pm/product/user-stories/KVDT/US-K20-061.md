### User Story

As a **Abrechnungsverantwortliche** (billing manager), I want to print the contents from the certificate overview (P20-060),
so that I have a physical record of the billing certificate overview for controlling and audit purposes.

### Compliance Context

The certificate overview (Scheinuebersicht) from P20-060 provides a controlling function for billing data. The ability to print this overview supports audit trails and allows practices to maintain physical documentation for review and compliance verification.

### Acceptance Criteria

**Scenario 1: Print certificate overview contents** [derived]
Given the certificate overview per [P20-060](../../compliances/KVDT/P20-060.md) is available
When the user selects the print function
Then the contents from [P20-060](../../compliances/KVDT/P20-060.md) are printed
