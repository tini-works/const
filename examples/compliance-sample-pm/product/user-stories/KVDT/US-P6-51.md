### User Story

As a **System**, The system must process eGKs with the test cost carrier VKNR 74799 for data capture but prevent transmission of this cost carrier in the billing file,
so that gematik test data does not corrupt real billing submissions to the KVs.

### Compliance Context

The cost carrier with VKNR 74799 exists in the KT master data file exclusively for gematik testing purposes. If transmitted in a real billing file, it would cause errors at the KV. The system must handle eGKs with this VKNR for data capture but block all billing-related outputs.

### Acceptance Criteria

**Scenario 1: eGK data with VKNR 74799 is processed**
Given a patient presents an eGK with cost carrier VKNR 74799
When the eGK is read
Then the software processes the eGK, including reading in and adopting the insured person's data

**Scenario 2: No form printing for VKNR 74799**
Given a case is associated with cost carrier VKNR 74799
When the user attempts to print or issue contract physician forms
Then the software does not support printing/issuance of contractual physician forms with the data of this cost carrier

**Scenario 3: VKNR 74799 excluded from billing file**
Given the billing file is being generated
When cases with cost carrier VKNR 74799 exist
Then the software ensures the cost carrier is not written to and transmitted in the billing file
