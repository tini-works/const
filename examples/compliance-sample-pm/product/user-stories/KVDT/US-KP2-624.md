### User Story

As a **Abrechnungsverantwortliche**, I want **to capture the type of disease as free text (FK 5079) during service documentation and transmit it with billing**,
so that **the disease type justification for genetic examination GOPen is properly documented per EBM provisions**.

### Compliance Context

EBM provisions for genetic examination GOPen (11233, 11511-11513, 11516-11518, 11521, 11522, 19421, 19424, 19451-19453, 19456) require the type of disease specification. Multiple entries must each result in a separate FK 5079 field in the billing file.

### Acceptance Criteria

**Scenario 1: Free text entry for type of disease**
Given the user is documenting a service requiring type of disease specification
When the user enters the "Type of disease"
Then the software enables free text entry in FK 5079

**Scenario 2: Type of disease transmitted in billing**
Given the user has specified the type of disease
When the billing is created
Then the software transmits the value in field 5079

**Scenario 3: Multiple entries transmitted as multiple fields**
Given the user makes multiple entries (count > 1) for "Type of disease"
When the billing file is generated
Then field 5079 is transmitted multiple times (count > 1)
