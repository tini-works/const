### User Story

As a **Abrechnungsverantwortliche**, I want **to capture and transfer the Ausstellungsdatum (FK 4102) for Muster 10 and Muster 10A referrals**,
so that **the temporal assignment of the treatment case in labor clearing is correctly documented**.

### Compliance Context

For in-vitro-diagnostic orders via Muster 10/10A, the Ausstellungsdatum is required for temporal assignment in labor clearing. Without it, billing records lack the necessary date reference, leading to processing errors at the KV.

### Acceptance Criteria

**Scenario 1: Ausstellungsdatum capture prompted**
Given a referral for in-vitro-diagnostic Auftragsleistungen via Muster 10 or Muster 10A is captured
When the user creates the billing record
Then the software prompts the user to capture the Ausstellungsdatum from the respective Muster

**Scenario 2: Future date validation**
Given the user is entering the Ausstellungsdatum
When the Ausstellungsdatum is later than the system date
Then the software rejects the entry with an error message

**Scenario 3: Ausstellungsdatum transferred in billing**
Given the user has entered a valid Ausstellungsdatum
When the billing is created
Then the software transfers the Ausstellungsdatum in field 4102
