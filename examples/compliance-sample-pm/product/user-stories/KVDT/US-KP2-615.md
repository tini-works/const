### User Story

As a **Abrechnungsverantwortliche**, I want **to specify at least one HGNC gene symbol (FK 5077), optionally one or more types of disease (FK 5079), and a valid ICD-10-GM code (FK 6001) when capturing GOPen 19424, 19453, and 19456[G-alpha]**,
so that **the billing justification includes the required examination type and disease coding per EBM provisions**.

### Compliance Context

EBM provisions for GOPen 19424, 19453, and 19456 require the HGNC gene symbol and ICD-10-GM coded disease type. The "Type of disease" in FK 5079 is optional but must be supported. Incomplete billing justifications result in rejected claims.

### Acceptance Criteria

**Scenario 1: At least one HGNC gene symbol required**
Given the user is capturing GOPen 19424[G-alpha], 19453[G-alpha], or 19456[G-alpha]
When the service is being documented
Then the software requires at least one "HGNC gene symbol" (FK 5077)
And the user may also enter multiple HGNC gene symbols

**Scenario 2: Type of disease optional but supported**
Given the user is capturing GOPen 19424[G-alpha], 19453[G-alpha], or 19456[G-alpha]
When the service is being documented
Then the "Type of disease" (FK 5079) does not need to be mandatorily captured
And the user must be given the ability to capture one or more "Types of disease"

**Scenario 3: HGNC gene symbol chains usable**
Given the user needs to enter multiple HGNC gene symbols
When entering data for the mentioned GOPen
Then HGNC gene symbol chains according to function [KP2-623] can be used
