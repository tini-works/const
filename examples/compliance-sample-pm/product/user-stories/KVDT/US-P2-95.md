### User Story

As a **Praxisadministrator**, I want to easily locate the encrypted KVDT billing file in the file system,
so that I can use the online billing services of the KVen without unnecessary difficulty.

### Compliance Context

Online billing services of the KVen require the user to locate and upload the encrypted KVDT billing file. Without clear documentation and system-side support for locating this file, users may fail to submit billing on time or submit incorrect files.

### Acceptance Criteria

**Scenario 1: Documentation of billing file location [derived]**
Given the PVS documentation (user manual, online help) is available
When the user looks for billing file location information
Then the documentation includes information about where in the file system the encrypted KVDT billing file is stored

**Scenario 2: System-side file location function [derived]**
Given the user needs to find the encrypted billing file
When the user invokes the file location function
Then the system supports the user by directly navigating to the billing file, invoking a file explorer with a correctly pre-set path, or providing direct access to the corresponding online help information
