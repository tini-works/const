### User Story

As a **Abrechnungsverantwortliche**, I want **to create separate Abrechnungsdatensätze when multiple referral forms exist for the same patient in the same quarter**,
so that **each referral is billed independently as required by the KVDT billing structure**.

### Compliance Context

A patient may present multiple referral forms (Überweisungsscheine) from different physicians or for different purposes in the same quarter. Each referral must be billable as a separate data record to maintain correct attribution of services to their respective referral context.

### Acceptance Criteria

**Scenario 1: Separate billing records for multiple referrals [derived]**
Given multiple referral forms of the same patient exist for the same quarter
When the user creates billing records
Then separate Abrechnungsdatensätze can be created for each referral
