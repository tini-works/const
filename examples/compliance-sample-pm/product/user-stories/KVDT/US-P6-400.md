### User Story

As a **Praxisadministrator**, I want the billing software to use the currently valid PLZ master data file from KBV at the beginning of each quarter,
so that postal code validations in the billing system are accurate and up to date.

### Compliance Context

The KBV provides a postal code (PLZ) master data file for validation purposes. Billing software must implement validations against this file to ensure correct patient and practice address data. Outdated postal code data leads to validation errors and potentially rejected billing records.

### Acceptance Criteria

**Scenario 1: PLZ validations implemented with current file** [derived]
Given a new quarter is beginning
When the billing software performs validations
Then validations with the currently valid postal code master data file of the KBV are implemented in the billing software
