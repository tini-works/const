### User Story

As a **System**, The system must deploy the EBM master data file for the following quarter as defined in the header validity period,
so that fee schedule data is applied to the correct billing quarter.

### Compliance Context

The EBM master data file delivered by the update deadline is intended for use in the following quarter. Deploying it in the wrong quarter would apply incorrect fee values and billing rules, causing systematic errors across all service documentation.

### Acceptance Criteria

**Scenario 1: EBM file deployed for correct quarter** [derived]
Given an EBM master data file has been delivered by the update deadline
When the system deploys the file
Then the file is used for the following quarter as defined in its header validity period
And it is not used for the current quarter's billing
