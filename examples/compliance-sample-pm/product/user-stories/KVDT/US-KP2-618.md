### User Story

As a **System**, The system must **require a valid ICD-10-GM code (distinct from Z01.7) with Diagnosis certainty when capturing genetic examination GOPen, and transmit both in the billing**,
so that **the disease type is properly coded per EBM provisions for billing transparency of genetic examinations**.

### Compliance Context

EBM billing provisions for genetic examination GOPen require the ICD-10-GM coded disease type with diagnosis certainty. The code Z01.7 is explicitly excluded. Without valid ICD-10-GM coding, billing claims for genetic services will be rejected by the KV.

### Acceptance Criteria

**Scenario 1: Valid ICD-10-GM code with diagnosis certainty required**
Given the user is capturing the relevant genetic examination GOPen
When the service is being documented
Then the software requires a valid ICD-10-GM code (distinct from Z01.7) with "Diagnosis certainty"

**Scenario 2: ICD-10-GM code and diagnosis certainty transmitted**
Given the user has provided the ICD-10-GM code and diagnosis certainty
When the billing is created
Then the software transmits the ICD-10-GM code in field 6001
And the Diagnosis certainty in field 6003
