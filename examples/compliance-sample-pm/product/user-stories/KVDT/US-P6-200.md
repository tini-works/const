### User Story

As a **Praxisadministrator**, I want the system to provide the valid Arztverzeichnis-Stammdatei (SDAV) for all users billing laboratory cases,
so that practice and physician number inventories remain current for laboratory referral processing.

### Compliance Context

The SDAV is required for practices billing laboratory cases (Muster 10 or 10A). It ensures that referring physician data is accurate, which is essential for correct laboratory referral attribution. Outdated physician directories lead to billing errors in laboratory workflows.

### Acceptance Criteria

**Scenario 1: SDAV available for laboratory billing users** [derived]
Given the user bills laboratory cases (Muster 10 or 10A)
When the user accesses the billing software
Then the valid Arztverzeichnis-Stammdatei (SDAV) based on data set description SDAV0308.nn is available
