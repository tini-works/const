### User Story

As a **Vertragsarzt**, I want **to transfer the Betriebsstättennummer of the specialist to whom I mediate an appointment as billing rationale**,
so that **I can bill GOPen 03008/04008 with the required BSNR documentation per § 295 Abs. 1 Satz 1 Nr. 3 SGB V**.

### Compliance Context

Physicians are obligated per § 295 Abs. 1 Satz 1 Nr. 3 SGB V to transmit the specialist's identifier when billing for mediated appointments. The BSNR is transferred in KVDT-Feld 5003 for direct assignment to the surcharge. Without this, appointment mediation surcharges (GOPen 03008/04008) cannot be properly justified.

### Acceptance Criteria

**Scenario 1: BSNR transfer for appointment mediation**
Given the user is mediating an appointment to a specialist
When the user specifies the specialist's Betriebsstättennummer
Then the software transfers the specified Betriebsstättennummer into the billing in Feld 5003

**Scenario 2: Multiple entry methods for BSNR**
Given the user needs to specify a specialist's Betriebsstättennummer
When the user chooses an entry method
Then the software allows the user to:
  - enter the Betriebsstättennummer as free text, or
  - copy and insert the Betriebsstättennummer from a search result (e.g. from [KBV_Kollegensuche]), or
  - adopt the Betriebsstättennummer from the data per [KP2-508]
