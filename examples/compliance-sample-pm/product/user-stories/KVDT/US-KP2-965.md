### User Story

As a **Vertragsarzt**, I want **the system to notify me when no remaining contingent exists for an approved psychotherapy in the current quarter so I can specify Pseudo-GOP 88130**,
so that **termination notifications are submitted correctly per the Psychotherapy Agreement and billing is compliant**.

### Compliance Context

Section 17 para. 7 of the Psychotherapy Agreement (Annex 1 BMV-Ae) mandates that a termination notification via Pseudo-GOP 88130 is submitted when an approved psychotherapy has no remaining contingent. Failure to comply risks incomplete billing data and regulatory non-conformance with KV billing requirements.

### Acceptance Criteria

**Scenario 1: Notification after service recording with no remaining contingent**
Given an approved psychotherapy exists in the current quarter
And the user has just recorded a service
When the system determines no remaining contingent exists for the approved psychotherapy
Then the software must notify the user about the specification of Pseudo-GOP 88130

**Scenario 2: Transfer of Pseudo-GOP 88130 in ADT billing**
Given the user has specified Pseudo-GOP 88130 after notification
When the ADT billing is generated
Then the software transfers the Pseudo-GOP 88130 in field 5001

**Scenario 3: Reminder during billing creation for unrecorded Pseudo-GOP 88130**
Given an approved psychotherapy has no remaining contingent
And no Pseudo-GOP 88130 has been entered
When billing creation is initiated
Then the software must remind the user about the recording and transmission of a "Termination notification with Pseudo-GOP 88130"

**Scenario 4: User forgoes transfer due to follow-up application**
Given an approved psychotherapy has no remaining contingent
And the user chooses to forgo the transfer of Pseudo-GOP 88130
When a follow-up application for the continuation of the psychotherapy has been or is being filed
Then the software must allow the user to forgo the transfer in the current quarter

**Scenario 5: User forgoes transfer due to interruption exceeding 6 months**
Given an approved psychotherapy has no remaining contingent
And the psychotherapy is interrupted for longer than 6 months
And an informal justification per section 17 para. 6 Psychotherapy Agreement is submitted
When the user chooses to forgo the transfer of Pseudo-GOP 88130
Then the software must allow the user to forgo the transfer
And the software provides the ability to record this in the patient documentation
