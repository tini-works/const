### User Story

As a **System**, the system must use the calculation time (in minutes) per Annex 3 of the EBM -- not the examination time -- when creating daily profiles for specified psychotherapeutic services,
so that daily profiles are calculated correctly per the EBM Valuation Committee resolution effective 1 July 2019.

### Compliance Context

The EBM Valuation Committee resolution (439th session, 19 June 2019) changed the calculation basis for psychotherapeutic service daily profiles from examination time to calculation time. Using the wrong time basis produces incorrect daily profiles, which could trigger audits or incorrect workload assessments. The EBM master file provides both values in separate XML elements.

### Acceptance Criteria

**Scenario 1: Use calculation time for daily profile**
Given the user creates a daily profile
When the profile includes psychotherapeutic GOPs 30931, 30932, 35140, 35141, 35150, 35151, 35152, 35401, 35402, 35405, 35411, 35412, 35415, 35421, 35422, 35425, 35431, 35432, or 35435 (with G-alpha suffixes)
Then the software uses the calculation time (in minutes) per Annex 3 of the EBM as the calculation basis (element `//zeitbedarf_liste/zeit/@V`)
AND does not use the examination time (element `//pruefzeit/@V`)
