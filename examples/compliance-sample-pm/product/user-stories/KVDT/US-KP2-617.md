### User Story

As a **Abrechnungsverantwortliche**, I want **to specify the type of disease coded per ICD-10-GM for GOPen 11302, 11303, 19402, and 32901-32918[G-alpha]**,
so that **the billing justification includes the required ICD-10-GM disease coding without HGNC or free-text disease type fields**.

### Compliance Context

EBM provisions for these GOPen require only the ICD-10-GM coded disease type. Fields FK 5070, FK 5077, and FK 5079 must not be transmitted. Transmitting incorrect fields or omitting the ICD-10-GM code causes billing rejections.

### Acceptance Criteria

**Scenario 1: ICD-10-GM code capture per KP2-618**
Given the user is capturing GOPen 11302[G-alpha], 11303[G-alpha], 19402[G-alpha], or 32901-32918[G-alpha]
When the service is being documented
Then all provisions of function [KP2-618] apply for ICD-10-GM code capture

**Scenario 2: FK 5070, FK 5077, FK 5079 not transmitted**
Given the above-mentioned GOPen are being billed
When the billing file is generated
Then the software does not transmit fields FK 5070, FK 5077, and FK 5079
