### User Story

As a **Praxisadministrator** (practice administrator), I want to apply the KBV change file to update the KT master data file by adding new records or overwriting existing ones,
so that cost carrier master data stays current with the latest KBV-published changes.

### Compliance Context

The KBV publishes change files to the KT master data file as needed. Applying these changes ensures that cost carrier data remains accurate for billing. If a VKNR in the change file is not present in the local file, the record is added; if present, it is overwritten. Without timely updates, billing may use outdated cost carrier information leading to rejected claims.

### Acceptance Criteria

**Scenario 1: Add new VKNR from change file**
Given a billing VKNR in the change file under `/kostentraeger/@V` is not present in the user's KT master data file
When the change file is applied
Then the corresponding change data record is added to the KT master data file

**Scenario 2: Overwrite existing VKNR from change file**
Given a billing VKNR in the change file under `/kostentraeger/@V` is present in the user's KT master data file
When the change file is applied
Then the corresponding KT master record is overwritten (replaced) by the change data record
