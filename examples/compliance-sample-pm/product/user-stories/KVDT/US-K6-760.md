### User Story

As a **Vertragsarzt** (contract physician), I want to view the essential contents of the EBM master data file in an appropriate manner,
so that I can look up fee schedule information directly within the software for billing decisions.

### Compliance Context

The EBM master data file contains the fee schedule information that governs billing of contract physician services. Displaying its contents allows physicians to make informed billing decisions. Without access to this data within the PVS, physicians would need to consult external sources, increasing the risk of billing errors.

### Acceptance Criteria

**Scenario 1: Display EBM master data file contents** [derived]
Given the EBM master data file is loaded in the software
When the user requests to view the EBM data
Then the essential contents of the EBM master data file are displayed in an appropriate manner
