### User Story

As a **Vertragsarzt**, I want **to transfer additional psychotherapy service data fields alongside billing data**,
so that **psychotherapy services are fully documented with the required supplementary fields per KVDT Datensatzbeschreibung**.

### Compliance Context

Psychotherapy billing requires additional data fields beyond standard billing data. The predefined fields (FK 4234-4257) must be transferable to ensure complete service documentation. Missing psychotherapy-specific data leads to incomplete billing records.

### Acceptance Criteria

**Scenario 1: Psychotherapy service fields transferable**
Given the user is billing psychotherapy services
When the user accesses the psychotherapy service documentation
Then the user has the option to transfer fields 4234, 4235, 4236, 4247, 4250, 4251, 4252, 4253, 4254, 4255, 4256, and 4257 as defined in [KBV_ITA_VGEX_Datensatzbeschreibung_KVDT]
