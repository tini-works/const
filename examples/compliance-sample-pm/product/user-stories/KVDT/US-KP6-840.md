### User Story

As the **System**, the system must **integrate and use the valid OPS master data file (SDOPS) provided by the KBV**,
so that **procedure coding complies with the valid version required by section 295 Abs. 1 SGB V**.

### Compliance Context

Section 295 Abs. 1 SGB V mandates that operations and procedures are coded using the valid version of the procedure code. The SDOPS represents this valid version. Using outdated or incorrect master data leads to billing rejections and non-compliance with statutory coding requirements.

### Acceptance Criteria

**Scenario 1: OPS master data file integration**
Given the KBV has provided the OPS master data file [SDOPS]
When the software is configured for procedure coding
Then the OPS master data file must be integrated into the software
And the current valid version must be in use
