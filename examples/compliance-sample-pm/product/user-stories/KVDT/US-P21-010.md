### User Story

As a **Vertragsarzt**, I want to mark GOPs with an ASV team number during service documentation,
so that ASV services are correctly attributed to the right team for billing purposes.

### Compliance Context

The contractual basis is section 6, para. 3 of the ASV-AV in conjunction with section 116b, para. 6, sentence 12 SGB V. Each GOP performed within ASV must be linked to the correct team number. Without this linkage, the KV cannot process the billing correctly, and the ASV services will be rejected.

### Acceptance Criteria

**Scenario 1: GOPs can be marked with ASV team number**
Given the user is recording services within the ASV framework
When the user documents a GOP
Then the software enables the user to mark the GOP with a defined ASV team number

**Scenario 2: ASV team number transmitted with GOP**
Given a GOP has been marked with an ASV team number
When the billing is generated
Then the software transfers the ASV team number assigned to the GOP in field FK 5100 (ASV team number of the contract physician)
