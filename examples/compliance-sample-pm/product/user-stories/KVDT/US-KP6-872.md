### User Story

As the **System**, the system must **ensure that for OPS code 5-622.5, a side localization (R or L) is always required and transmitted in FK 5041**,
so that **billing complies with the Bewertungsausschuss decision per section 87 Abs. 1 Satz 1 SGB V, effective January 1, 2009**.

### Compliance Context

The Bewertungsausschuss decision in its 167th session mandates side localization for OPS 5-622.5 in contractual physician care, overriding the general OPS master data side specification. Without this special handling, the billing file would lack the required side localization, causing rejection. The provisions of KP6-871 do not apply for this code.

### Acceptance Criteria

**Scenario 1: Side localization always required for OPS 5-622.5**
Given the user documents OPS code 5-622.5
When the code is prepared for billing
Then the software must require a side specification (R or L)
And the side specification is transmitted in FK 5041
